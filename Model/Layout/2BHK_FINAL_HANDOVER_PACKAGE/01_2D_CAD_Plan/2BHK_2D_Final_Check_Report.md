2. **Construction Raised Plinths:** All container plinths must be raised **450mm** above ground Z-plane, finished in native red laterite masonry.
3. **Monsoonal Runoff Setbacks:** Drainage safety river gravel strips ($300\text{mm}$ width) must wrap the plinths, draining directly to decentralized bioswales.
4. **Utility Pad Coverages:** bio-digester STP, rainwater cisterns, and greywater reuse collection pads must be fully screened using vertical timber slat screen structures.
