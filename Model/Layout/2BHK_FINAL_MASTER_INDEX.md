## 🛠️ 4. How-To Operation Guides

### A. How to Run the Ruby SketchUp Generator
To programmatically regenerate the entire upgraded 3D SketchUp model:
1. Open SketchUp 2023.
2. Go to **Extensions -> Ruby Console**.
3. Type the following command and press Enter:
   ```ruby
   load "C:/Users/INTEL/Desktop/GOA/WEBSITE/Model/Layout/reserve_varde_2bhk_upgraded_generator.rb"
   ```
4. The script will automatically clear the current space, build the multi-level terraced terrain, assemble all container wings, place solar, utility, landscaping, courtyard pools, and automatically export the 10 Full-HD native render images into the `Reserva_Varde_2BHK_Final_Renders/` directory.

### B. How to Run the Lumion Compiler & Exporter
To programmatically regenerate the 100% clean, Lumion-ready model and Collada DAE:
1. Open SketchUp 2023.
2. Open the **Extensions -> Ruby Console**.
3. Type the following command and press Enter:
   ```ruby
   load "C:/Users/INTEL/Desktop/GOA/WEBSITE/Model/Layout/reserve_varde_2bhk_lumion_generator.rb"
   ```
4. The compiler will clean all loose lines, group the model onto isolated, standard layers, map all faces to unique `RV_` prefix materials, and automatically write `Reserva_Varde_2BHK_Lumion_Ready.skp` and `Reserva_Varde_2BHK_Lumion_Export.dae` into the `Lumion_Export_Package/` folder.

### C. How to Import and Setup inside Lumion
1. Launch **Lumion 12+**. Create a new scene using the **Tropical / Forest Hills** environment.
2. Click **Import**, browse and load `Reserva_Varde_2BHK_Lumion_Export.dae`.
3. Set the imported model coordinates to `[0, 0, 0]`, keeping scale at **1.0**.
4. Go to **Reflection Effects** inside your Photo view, and place manual **Reflection Planes** on the courtyard plunge pool, concrete reflecting channel, and main sliding glass doors facade (`RV_Glass_Dark_Reflective`).
5. Open the **Material Editor**, double-click material slots, and map them to matching Lumion 3D presets (e.g. Teak Wood 3D, Rough Sandstone for Laterite, Pure Glass, and 3D Grass) using the [Lumion_Material_Map.md](file:///C:/Users/INTEL/Desktop/GOA/WEBSITE/Model/Layout/Lumion_Export_Package/Lumion_Material_Map.md) slider recommendations.
6. Replace background placeholders (`RV_Landscape_Shrubs`) with realistic Lumion coconut palms, banana trees, and tall teak trees. Cluster broad broadleaf sword ferns and elephant ears at the laterite plinth bases.
7. Configure ceiling interior spotlights ($2700\text{K}$) and set `RV_Emissive_Light` power to **20.0** for warm evening illumination.

## ⚖️ 5. Pre-Construction Engineering & Architect Review

> [!IMPORTANT]  
> All drawings, models, and technical parameters developed in this package represent high-fidelity concept designs. Before commencing site grading, foundation pouring, or container modular assembly in South Goa, the developer must submit this package to a licensed local Goan architectural and structural engineering office for a professional review of:
> 1. **Contour & Geotechnical Audit:** bored concrete foundation piles must be calculated to anchor the heavy red laterite plinths and modular steel castings through Goan clay forest soils.
> 2. **Cyclonic Wind-Load Certification:** Solar panels mounts and container sloped roofs must be certified under BIS IS-875 (Part 3) wind-load calculations to safely withstand peak $50\text{ m/s}$ (180 km/h) monsoonal wind-lift.
> 3. **Hydrology & Runoff Capacity:** Gutters and bioswales must be audited to ensure complete, rapid monsoonal drainage ($347\text{ Liters/minute}$ peak load).
> 4. **Municipal Clearances:** Complete CRZ forest setback mapping and Panchayat building sanctions must be formally secured for the bio-digester sewage treatment and underground rainwater harvest cylinder layouts.
