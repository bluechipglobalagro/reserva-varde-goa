## 🛠️ 2. How-To Operation Summary

### How to Regenerate the Models
1. **Upgraded SketchUp Model:** Load `reserve_varde_2bhk_upgraded_generator.rb` in SketchUp’s Ruby console to compile the parametric model and automatically export the 10 Full-HD native render images.
2. **Lumion Cleaned Model & DAE:** Load `reserve_varde_2bhk_lumion_generator.rb` in SketchUp's Ruby console to compile the 100% solid, cleaned model and DAE mesh.

### How to Render in Lumion
1. **Load Mesh:** Import `Reserva_Varde_2BHK_Lumion_Export.dae` in Lumion 12+ ( Tropical/Forest Hills scene), keeping XYZ at `[0,0,0]` and scale at `1.0`.
2. **Assign Materials:** Map all material slots (prefixed with `RV_`) to matching high-end Lumion 3D presets (e.g. Teak Wood 3D, Rough Sandstone for Laterite, Pure Glass, and 3D Grass) using `Lumion_Material_Map.md` parameters.
3. **Set Reflections:** Add reflection planes to the plunge pool, reflecting channel, and main glass facade (`RV_Glass_Dark_Reflective`).
4. **Place foliage:** Replace background tree placeholders with high-fidelity Lumion coconut palms, banana trees, and tall teak trees. Cluster broad-leaf ferns and elephant ears at the laterite plinth bases.
5. **Configure Spotlights:** Setup interior ceiling spotlights ($2700\text{K}$) and set `RV_Emissive_Light` power to **20.0** for evening path light illumination.
6. **Apply Effects:** Setup the pre-saved views (Hero Wide, Dusk Arrival, Drone Aerial, Courtyard pool, etc.) and apply the recommended effects stack (Real Skies, Hyperlight at 75%, Soft Shadows, 2-Point Perspective, and subtle Depth of Field).
